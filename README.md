# SPIDER (수동이)
안녕하세요. 개발자 김삘롱 입니다.  
많은 분들의 지지 덕분에 수동이를 이렇게 배포하게 되었습니다.   
  
## 수동이 핵심 기능
#### 1. 익절손절이 가능한 거미줄 트레이딩
#### 2. 실시간 뉴스 추적 알림
#### 3. 실시간 고래들의 체결정보와 지갑 이동 추적
#### 4. GPT를 통해 번역된 정제된 비트코인 관점

## 스마트한 코인 트레이더의 필수 아이템
### 비트겟, OKX, 바이비트, 빙엑스를 지원합니다.
